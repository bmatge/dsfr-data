# specs/

Specifications et pages de demonstration des composants dsfr-data.

## Contenu

- `charts/` -- Specifications par type de graphique (bar, line, pie, radar, map, gauge, scatter)
- `components/` -- Specifications par composant (`dsfr-data-source`, `dsfr-data-query`, `dsfr-data-normalize`, `dsfr-data-facets`, `dsfr-data-search`, `dsfr-data-list`, `dsfr-data-display`, `dsfr-data-chart`, `dsfr-data-kpi`)
- `index.html` -- Page d'accueil des specifications

## Usage

Ces pages servent de reference pour le comportement et les attributs de chaque composant. Elles sont accessibles en dev via le serveur Vite (ex: `/specs/index.html`).
