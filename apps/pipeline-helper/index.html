Pipeline Helper

Comment utiliser le Pipeline Helper
  1. Ajouter des composants avec les boutons de la barre d'outils.
  2. Connecter : glissez d'un cercle de sortie (droite) vers un cercle d'entree (gauche).
  3. Configurer : remplissez les champs dans chaque noeud.
  4. Executer : cliquez "Executer" pour voir les donnees traverser le pipeline.
  5. Inspecter : cliquez sur un noeud pour voir ses donnees dans le panneau de droite.
Pipeline type : Source (fetch) → Query (transforme) → Chart / List / KPI (affiche).
